Como usar

<link rel="stylesheet" href="/shared/shared.css">

/* tokens disponíveis */
color: var(--bnd-ink);        background: var(--bnd-lime);
font-family: var(--bnd-display);  /* Aeion Mono */
font-family: var(--bnd-body);     /* Inter */
border: var(--bnd-rule);          /* 1px dashed ink */
border-radius: var(--bnd-radius); /* 16px */

Regra de ouro: display = Aeion caps (600/700/800) · corpo = Inter (400/700) · flat sempre — sem gradiente 3D, sem sombra mole, sem card fofo.
